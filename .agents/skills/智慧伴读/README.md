# 🦐 智慧伴读 · 阅读伴侣系统

> 不是替人读书，而是伴人读懂。

基于花叔《阅读的方法》改良版构建的**分层阅读系统**，帮你从「读完一本书」升级到「真正读懂一本书」。

---

## 🌟 核心能力

### 三级深度模式

| 模式 | 流程 | 适合场景 | 预计耗时 |
|------|------|---------|---------|
| 🏹 **轻量** | 狩猎法 → 苏格拉底追问 | 快速找答案、定位书中内容 | 15-30 分钟 |
| 🧠 **标准** | 狩猎法 → 费曼法 | 搞懂一个概念、暴露模糊地带 | 30-60 分钟 |
| ⚔️ **深度** | 狩猎法 → 费曼法 → 华山论剑 | 验证核心观点是否经得起推敲 | 60-120 分钟 |

### 输出物

- 📝 Markdown 结构化阅读笔记
- 🎨 HTML 可视化阅读卡片（交互式信息图）
